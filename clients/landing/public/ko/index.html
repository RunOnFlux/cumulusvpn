작동 방식

계정도, 저희 소유의 서버도 없습니다. 오직 체인뿐입니다.

당신의 WireGuard 키가 곧 당신의 신원입니다. 결제는 Flux 블록체인에 기록된 사실이며, 모든 게이트웨이가 각자 독립적으로 이를 읽습니다. 저희 회사가 내일 사라지더라도, 이미 배포된 게이트웨이는 계속 작동합니다.

1단계

생성 & 연결

앱이 사용자 기기에서 WireGuard 키 쌍을 생성하고 가장 가까운 데이터센터 게이트웨이에 등록합니다. 무료 등급은 서버 측에서 100 KB/s로 제한됩니다 — 몇 초 만에 연결되며, 가입 절차는 없습니다.

pubkey → POST /v1/enroll → tunnel up
2단계

FLUX로 결제

결제 주소로 20 FLUX(약 $0.99)를 보내고, 거래 메시지에 본인 키의 코드를 함께 적어주세요. 탭 한 번으로 내용이 미리 채워진 지갑이 열립니다.

memo: CVPN1:<code(pubkey)>
3단계

어디서든 잠금 해제

모든 게이트웨이가 체인을 스캔해 ~1 분 안에 당신의 키를 전체 속도로 전환합니다 — 모든 서버에서 동시에, 30 일 동안. 활성화도 로그인도 필요 없습니다. 미리 결제해 개월 수를 쌓아둘 수 있습니다.

chain → paid_until[key] += 30d

연결 버튼 하나로, 당신이 가진 모든 화면에.

웹, 데스크톱, 모바일 모두 동일한 세련된 클라이언트 — 하나의 디자인 언어와 하나의 키를 공유합니다. 결제는 웹이나 데스크톱에서 하세요. 스마트폰은 체인을 통해 1분 안에 자동으로 잠금 해제됩니다.

웹에서 설정

브라우저에서 키를 생성하고 WireGuard 앱으로 바로 가져올 수 있는 설정 파일과 QR 코드를 제공합니다. FLUX로 결제하는 업그레이드 페이지도 여기에 있습니다. 설정이나 결제 모두 설치가 필요 없습니다.

저희가 가진 적 없는 것은 내드릴 수도 없습니다.

대부분의 VPN은 '로그를 남기지 않는다'는 약속을 그대로 믿어달라고 합니다. CumulusVPN은 애초에 신뢰해야 할 대상 자체를 없앴습니다. 기록할 계정도 없고, 사용자 데이터베이스도 없으며, 전체 게이트웨이를 소유한 중앙 서버도 없습니다.

로그 없음, 항상

게이트웨이는 키별로 등급을 적용하고 트래픽을 라우팅합니다 — 당신이 누구인지, 무엇을 보는지에 대한 어떤 정보도 저장되지 않습니다. DNS는 터널을 거쳐 게이트웨이 자체 리졸버로 전달됩니다.

계정 없음

이메일도, 비밀번호도, 카드도 필요 없습니다. 당신의 WireGuard 키가 유일한 신원이며, 결제는 해시에 연결된 익명의 FLUX 송금입니다 — 당신 개인에게 연결되지 않습니다.

오픈소스

앱과 게이트웨이 모두 공개되어 있습니다. 코드를 직접 읽고, 직접 빌드하거나, 자신만의 게이트웨이를 운영할 수도 있습니다 — 결제 및 권한 부여 프로토콜이 완전히 문서화되어 있습니다.

실제 데이터센터 노드

종료 노드는 Flux 데이터센터 인프라에서 운영됩니다 — 가정용 봇넷이 아닙니다. 안정적인 IP, 실제 대역폭, 네트워크 전체 6,500개 이상의 노드.

분산형 게이트웨이 함대

클라이언트는 Flux 네트워크 자체에서 활성 게이트웨이를 직접 탐색하며, 서명되어 앱에 내장된 대체 목록도 갖추고 있습니다. 압수하거나 차단하거나 정지시킬 단일 지점이 없습니다.

내부는 WireGuard

현대적이고, 감사받았으며, 빠릅니다. 설정 파일을 직접 만질 일은 전혀 없습니다 — 앱이 키, 터널, 장애 조치를 모두 처리합니다. WireGuard는 배선이고, 사용자 경험은 버튼 하나입니다.
