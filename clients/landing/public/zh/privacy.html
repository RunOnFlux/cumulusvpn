隐私政策

最近更新：2026年7月16日 · 生效日期：2026年7月16日

本译文仅为方便阅读而提供；cumulusvpn.com/privacy 上的英文版本为准。

CumulusVPN 是一项去中心化的 VPN 服务。本政策用通俗 的语言，准确说明我们收集什么、不收集什么。之所以简短，是因为该服务在设计上 几乎不收集任何信息。如果本文中有任何说法有误，那是我们会修复的一个漏洞——架构 已在 cumulusvpn.com 及我们的源代码中公开说明。

1. 摘要（一屏看完整份政策）

  • 无账号。无邮箱。无密码。无电话号码。你从不需要向我们透露身份。
  • 无活动日志。我们不会将你访问的网站、DNS 查询、流量内容、 带宽历史或连接时间戳记录到任何持久化存储中。
  • 你的身份是一把加密密钥，在你自己的设备上生成。我们永远看不到你的 私钥，只能看到公钥——一串看似随机、与你本人没有任何关联的字符串。
  • 网关仅在 RAM 中保存对等状态。当服务器重启时，该状态即消失。 没有流量数据库可供查封、传票调取或泄露。
  • 付款是可选的，并通过公共区块链上的 FLUX 加密货币完成。我们 不运营该区块链，不收取银行卡信息，也永远无法从付款中得知你的姓名。
  • 无追踪器，无广告 SDK，无可识别你身份的分析工具。应用中不包含 任何第三方广告代码或行为追踪代码。
  • 我们不是数据生意。我们没有可供出售的用户数据，也不出售数据。

2. 我们是谁

CumulusVPN（"我们"）是由 CumulusVPN 团队运营的 VPN 服务，构建于 Flux 去中心化 云网络之上（runonflux.io）。VPN 出口服务器 （"网关"）以容器化应用的形式，运行在由独立、经过 KYC 验证的数据中心 运营商所运营的 Flux 数据中心节点上。我们发布该服务；其底层算力是去中心化的。

就欧盟/英国《通用数据保护条例》（GDPR）而言，在下列信息在极其有限的范围内可能构成 "个人数据"的情况下，CumulusVPN 运营主体即为数据控制者。我们所依据的合法依据是我们在提供功能完善、 抗滥用的 VPN 服务方面的正当利益（第 6(1)(f) 条）；就付款而言，则是为履行你所请求的服务 （第 6(1)(b) 条）。由于该服务在设计上就是匿名的，在大多数情况下我们并不持有 任何可与你关联的数据。

3. 我们不收集的内容

我们想说清楚，因为大多数 VPN 都会把这一点藏起来。CumulusVPN 不会收集、 记录或存储以下任何内容：

  • 你的姓名、电子邮箱、电话号码或任何账号凭证——因为根本没有账号。
  • 你的原始 IP 地址存入任何持久化日志。
  • 你连接到的网站、域名或服务。
  • 你的 DNS 查询（DNS 通过隧道为你解析，且不会被记录）。
  • 你的流量内容、时间或大小，存入任何持久化数据库。
  • 与你相关联的带宽历史、会话历史或连接时间戳历史。
  • 设备广告标识符（IDFA / GAID）——我们不请求也不使用它们。
  • 你设备的位置数据（精确或大致的 GPS）。你所经由连接的国家 是你对服务器的选择，而非对你设备位置的读取。
  • 通讯录、照片、文件、麦克风、摄像头或任何设备传感器。

我们不使用第三方广告网络，也不使用会为你建立画像的分析产品 （例如广告技术 SDK）。应用出厂即不含任何行为追踪器。

我们的网站（cumulusvpn.com 和 vpn.cumulusvpn.com）使用 Cloudflare Web Analytics，一项无 Cookie 的汇总统计服务：它不设置任何 Cookie，不在你的设备上存储 任何内容，也无法识别或建立单个访客的画像。它只告诉我们页面浏览次数， 而不是具体的人。应用中不包含任何形式的分析工具。

4. 技术上必然涉及的内容（以及为什么这不构成你的画像）

如果没有一些数据实时流经其中，VPN 就无法运作。以下就是全部这些数据， 以及为什么它们都不会成为关于你的记录。

4.1 你的公钥（假名标识符）

你的设备在首次启动时会在本地生成一对 WireGuard 密钥。私钥永远 不会离开你的设备——甚至不会传给我们。只有公钥会被发送到 网关，以便它能够为你加密的隧道选路。公钥是一个看似随机的 32 字节值。它 不与你的姓名、邮箱或硬件绑定，你也可以随时重新生成它。它是一个假名， 而非身份。

4.2 网关上的内存中对等状态

在你连接期间，网关仅在易失性内存（RAM）中保存路由数据包 所需的最少信息：你的公钥、为其分配的临时内部隧道 IP（例如 10.8.x.y），以及用于执行速度分级限制的字节计数器。该状态永远不会被写入 磁盘，不会传输给任何中央服务器，并且会在网关进程重启 或你断开连接时被清除。不会由此生成任何历史日志。

4.3 你的真实 IP 地址（短暂存在）

为了传输你的流量，网关在网络层必然会在当下 看到你连接来源的 IP 地址——任何互联网服务器都是这样运作的。CumulusVPN 不会将该地址记录到任何 持久化存储中，也不会将其与你的浏览行为关联。当你的会话结束时，它不会被保留。

4.4 DNS

DNS 查询会通过加密隧道代你解析，使你的本地网络和 ISP 无法看到它们。我们不保留 DNS 查询日志。

4.5 你的应用如何联系网关

你的应用会直接调用每个网关的控制 API，以发现网关并读取其状态。 这些响应会使用网关的 ed25519 密钥签名，并由应用进行验证， 因此即便控制通道本身只是普通连接，篡改也会被检测出来——完整性保证 来自签名，而非证书。你真正的互联网流量是分开的， 并在你的设备与网关之间使用 WireGuard 进行端到端加密。

4.6 网关运营方能与不能看到什么

网关运行在 Flux 数据中心节点上。和任何 VPN 出口一样，观察自身服务器的运营方 能够实时看到加密的流量元数据（某个对等端正在发送流量），但看不到我们生成的 持久化日志，也看不到隧道内容。对于不希望任何单一服务器同时知道其 IP 和目的地的用户，CumulusVPN 提供了可选的多跳模式：你的流量会 经过两个网关路由，使任何单一服务器都无法同时知道你是谁以及你要 去哪里。多跳默认关闭，并在应用中如实说明。

5. 付款（仅当你升级到高级版时）

CumulusVPN 提供免费层，无需付款，也无需任何身份识别信息。

如果你选择升级到高级版（更高速度），你需要在 Flux 公共区块链上以FLUX 加密货币 付款。要点如下：

  • 付款在网页或桌面端进行，而非在移动应用内。移动应用 不包含购买页面；它们只显示你当前的等级。
  • 你从自己的钱包向我们公布的付款地址发送 FLUX，并附上一段 由你公钥派生出的简短备注，以便网络能将款项计入正确的密钥。我们 不会收集你的银行卡号、账单地址或姓名。
  • 区块链交易天然是公开的。包括我们在内的任何人，都能在 公共账本上看到某笔金额的付款已连同某个特定备注发送到了我们的地址。该账本 由 Flux 区块链运营，而非由我们运营。我们不会将该笔交易与你的真实身份关联，且该备注 是从你的公钥单向派生出来的——它不会透露你的密钥或流量。请理解， 向公共区块链发布交易本质上就是公开的；请相应地自行留意你钱包的隐私。
  • 权益（某密钥是否为高级版、以及有效期至何时）是根据 公共区块链确定性计算得出的，且仅与公钥绑定。我们不会存储任何写有你姓名的单独客户账单记录， 因为我们根本没有你的姓名。

如果你之后通过 Apple App Store 或 Google Play 提供的应用内法币支付选项付款 （如果我们添加了此选项），该交易将由 Apple 或 Google 按照其自身的隐私政策 处理；我们只会收到授予高级版所需的确认信息，而不会收到你的付款卡数据。

6. 支持与自愿联系

如果你给我们发邮件（寻求支持或举报滥用），我们会收到你在邮件中写下的一切——例如 你的邮箱地址和消息内容。我们仅将其用于回复你，并且保留时间不会超过解决该 问题所需的期限。你无需表明身份即可使用本服务；联系支持完全是自愿的。

可选的、需主动开启的崩溃报告：如果——且仅当——你明确启用了崩溃报告， 一份包含技术诊断信息（如应用版本、设备型号、操作系统版本和堆栈跟踪）的崩溃报告 可能会被发送以帮助我们修复错误。该功能默认关闭，不包含任何浏览活动，且可随时关闭。

7. Cookie 与网站

CumulusVPN 网站（cumulusvpn.com）是一个静态网站，用于提供下载、文档、 付款页面，并托管我们的已签名服务器目录。它不使用广告 Cookie 或 第三方行为追踪器。所使用的任何 Cookie 均为纯功能性用途（例如记住 浅色/深色主题）。

8. 数据共享与披露

我们不出售、出租或交易个人数据——因为我们几乎没有可供出售的数据。

由于我们不保留任何活动日志或用户身份信息，针对某用户浏览行为的数据请求， 我们没有任何东西可以交出。如果我们收到执法部门或法院提出的合法有效请求， 我们只能提供我们实际持有的信息——其中不包括用户身份、浏览历史、 DNS 查询或流量内容，因为这些记录根本不存在。在合法且适当的情况下，我们打算 发布一份说明所收到请求情况的透明度报告。

我们不会将用户画像转移给广告商、数据经纪商或分析服务提供商，因为我们 根本不建立用户画像。

9. 国际数据传输

CumulusVPN 是一个覆盖全球的网关网络。当你连接时，你的流量会从 你所选择的服务器所在国家出网。由于我们不会将个人数据汇集到中央 数据库中，因此不存在需要保护的个人数据存储的跨境传输。出口国家 的选择完全由你决定，且每个会话都会重新选择。

10. 你的权利（GDPR / UK GDPR / CCPA 及类似法规）

根据你所在的地区，你可能拥有访问、更正、删除或转移你个人数据的权利， 反对或限制处理的权利，以及（在加利福尼亚州）知晓被收集了哪些信息、且不因行使这些权利而受到歧视的权利。 由于本服务是匿名的：

  • 访问 / 数据可携带权：我们通常没有任何与你相关联的个人数据可以提供。 如果你曾经发邮件给支持团队，你可以索取那次通信的副本。
  • 删除：没有任何与你身份绑定的内容需要我们从网关中删除；对等状态本身就是临时的 RAM 数据，会在断开连接/重启时清除。你可以随时在设备上删除你的密钥，这将彻底切断这条假名链接。
  • 不出售 / 不共享：我们不出售也不共享个人数据，因此没有什么可供选择退出的。

如需行使任何权利，请发邮件至 info@cumulusvpn.com。我们可能无法在我们的 （不存在的）记录中识别出你，这恰恰是因为该服务本身就是匿名设计——这是设计使然，而非逃避。你同样有权向当地的数据保护机构提出投诉。

11. 儿童

CumulusVPN 并非面向 13 岁以下儿童（或你所在国家/地区数字同意的最低年龄）。 我们不会故意收集儿童的个人数据。由于我们不会从任何人那里收集身份识别数据，我们 无法判断用户的年龄；本服务面向大众用户，并未针对儿童进行营销。

12. 安全

你的设备与网关之间使用 WireGuard® 协议（现代、经过审计的密码学技术）进行 端到端加密。你的私钥永远不会离开你的设备。服务器端的对等状态 仅保存在内存中。应用通过直接连接访问每个网关的控制 API，其响应 经 ed25519 签名并由应用验证，因此服务器目录和网关响应的真实性由数字签名——而非传输层证书——来保护。

13. 本政策的变更

如果我们变更本政策，我们会更新上方的"最近更新"日期，并在 cumulusvpn.com/privacy 发布新版本。重大变更将被 特别标注。由于我们不维护任何邮件列表，本网站是当前政策的 权威来源。
