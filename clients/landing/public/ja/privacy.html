CumulusVPN は分散型の VPN サービスです。本ポリシーは、私たちが何を収集し、何を収集しないかを 平易な言葉で正確に説明します。サービスはほとんど何も収集しないよう設計されているため、内容は短くなっています。もし本ポリシーの記述に誤りがあれば、 それは私たちが修正すべきバグです——アーキテクチャは cumulusvpn.comおよびソースコードで公開されています。

1. 概要（1 画面で読めるポリシー全体）

  • アカウント不要。メール不要。パスワード不要。電話番号不要。あなたが私たちに身元を明かす必要は一切ありません。
  • アクティビティログなし。あなたが訪れたウェブサイト、DNS クエリ、トラフィックの中身、 帯域使用履歴、接続タイムスタンプを、いかなる永続的な保存先にも記録しません。
  • あなたの識別情報は暗号鍵であり、あなた自身のデバイス上で生成されます。私たちがあなたの秘密鍵を 見ることは決してありません。見えるのは公開鍵だけであり、それはあなたと結びつかない、ランダムに見える文字列です。
  • ゲートウェイはピア状態を RAM のみに保持します。サーバーが再起動すると、その状態は消えます。 差し押さえたり、召喚状で提出させたり、漏洩したりできるトラフィックデータベースは存在しません。
  • 支払いは任意であり、公開ブロックチェーン上の FLUX 暗号資産で行われます。私たちは そのブロックチェーンを運営しておらず、カード情報を受け取ることもなく、支払いからあなたの名前を知ることもありません。
  • トラッカーなし、広告 SDK なし、あなたを特定できる分析ツールなし。アプリには サードパーティの広告コードや行動追跡コードは一切含まれていません。
  • 私たちはデータビジネスではありません。売るべきユーザーデータを持っておらず、データを販売することもありません。

2. 私たちについて

CumulusVPN（「私たち」）は、CumulusVPN チームが運営する VPN サービスであり、Flux の分散型 クラウドネットワーク（runonflux.io）上に構築されています。VPN のエグジットサーバー （「ゲートウェイ」）は、独立した KYC 認証済みのデータセンター運営者が運用する Flux データセンターノード上で、 コンテナ化されたアプリとして稼働しています。私たちがサービスを提供し、その基盤となる計算資源は分散化されています。

EU/英国一般データ保護規則（GDPR）の適用上、以下の情報が「個人データ」に該当するごく限られた範囲において、 CumulusVPN の運営主体がデータ管理者となります。私たちが依拠する法的根拠は、機能的で悪用に強い VPN を提供することに関する 私たちの正当な利益（第 6 条 1 項 (f)）、および支払いについては、あなたが要求したサービスの履行 （第 6 条 1 項 (b)）です。サービスが匿名性を前提に設計されているため、ほとんどの場合、私たちは あなたに紐づけられるデータを一切保有していません。

3. 私たちが収集しないもの

私たちはこれを明確にしたいと思います。多くの VPN はこの点をあいまいにしているからです。CumulusVPN は、以下のいずれも収集、 記録、保存しません：

  • あなたの氏名、メールアドレス、電話番号、あるいはいかなるアカウント認証情報も——そもそもアカウントが存在しません。
  • あなたの発信元 IP アドレスを、いかなる永続的なログにも記録しません。
  • あなたが接続するウェブサイト、ドメイン、サービス。
  • あなたの DNS クエリ（DNS はトンネル経由であなたのために解決され、記録されません）。
  • あなたのトラフィックの内容、タイミング、サイズを、いかなる永続的なデータベースにも記録しません。
  • あなたに紐づく帯域使用履歴、セッション履歴、接続タイムスタンプ履歴。
  • デバイス広告識別子（IDFA / GAID）——私たちはこれらを要求も利用もしません。
  • あなたのデバイスの位置情報（GPS、正確なものも大まかなものも含む）。あなたが経由して接続する国は あなたが選んだサーバーであり、デバイスの位置情報を読み取った結果ではありません。
  • 連絡先、写真、ファイル、マイク、カメラ、その他いかなるデバイスセンサーも。

私たちはサードパーティの広告ネットワークを使用せず、あなたのプロファイルを構築するような分析製品 （広告テック SDK など）も使用しません。アプリは行動トラッカーを一切搭載せずに出荷されています。

私たちのウェブサイト（cumulusvpn.com および vpn.cumulusvpn.com）は、 Cookie を使わない集計型の計測サービスである Cloudflare Web Analytics を利用しています。Cookie を設定せず、あなたのデバイスに 何も保存せず、個々の訪問者を特定・プロファイリングすることもできません。分かるのはページの閲覧数であり、 個人ではありません。アプリにはいかなる分析機能も含まれていません。

4. 技術的に関与しているもの（そしてなぜそれがあなたのプロファイルにならないのか）

VPN は、何らかのデータがリアルタイムで通過することなしには機能しません。ここではその全体と、 それがなぜあなたに関する記録にならないのかを説明します。

4.1 あなたの公開鍵（仮名の識別子）

あなたのデバイスは初回起動時にローカルで WireGuard の鍵ペアを生成します。秘密鍵はあなたのデバイスから 決して外に出ません——私たちに届くこともありません。ゲートウェイに送られるのは公開鍵のみで、 これはあなたの暗号化トンネルの経路を決めるために使われます。公開鍵はランダムに見える 32 バイトの値です。 あなたの氏名、メール、ハードウェアとは結びついておらず、いつでも再生成できます。それは仮名であり、 識別情報ではありません。

4.2 ゲートウェイ上のメモリ内ピア状態

接続している間、ゲートウェイは——揮発性メモリ（RAM）のみに——パケットをルーティングするために 必要最小限の情報を保持します：あなたの公開鍵、それに割り当てられた一時的な内部トンネル IP（例： 10.8.x.y）、そしてあなたの速度階層を適用するために使われるバイトカウンターです。この状態はディスクに 書き込まれることは決してなく、いかなる中央サーバーにも送信されず、ゲートウェイのプロセスが 再起動されるか、あなたが切断した時点で消去されます。ここから作られる履歴ログは存在しません。

4.3 あなたの実 IP アドレス（一時的）

あなたのトラフィックを運ぶために、ゲートウェイはその瞬間、ネットワーク層で、あなたの接続がどの IP アドレスから 来ているかを必然的に把握します——これはあらゆるインターネットサーバーの仕組みです。CumulusVPN はこのアドレスを、いかなる永続的な 保存先にも記録せず、あなたの閲覧行動と紐づけることもありません。セッションが終了すると、それは保持されません。

4.4 DNS

DNS の名前解決は、あなたのローカルネットワークや ISP がそれを見ることができないよう、暗号化された トンネルを通じてあなたに代わって行われます。DNS クエリのログは保存しません。

4.5 アプリはどのようにゲートウェイに到達するか

あなたのアプリは、各ゲートウェイの制御 API を直接呼び出すことでゲートウェイを発見し、その状態を読み取ります。 それらの応答はゲートウェイの ed25519 鍵で署名され、アプリによって検証されるため、 制御チャネル自体は平文の接続であっても、改ざんは検出されます——完全性の保証は 証明書ではなく署名に由来します。あなたの実際のインターネットトラフィックは別物であり、 あなたのデバイスとゲートウェイの間でWireGuard によりエンドツーエンドで暗号化されます。

4.6 ゲートウェイ運営者が見られるもの・見られないもの

ゲートウェイは Flux のデータセンターノード上で稼働しています。他の VPN エグジットと同様、自分のサーバーを観察している 運営者は、リアルタイムの暗号化されたフロー・メタデータ（あるピアがトラフィックを送信していること）を見ることはできますが、私たちが作る 永続的なログや、トンネルの中身を見ることはできません。単独のサーバーに自分の IP と行き先の両方を知られたくないユーザーのために、CumulusVPN は任意のマルチホップモードを提供しています。 あなたのトラフィックは 2 つのゲートウェイ経由でルーティングされ、単独のサーバーがあなたが誰で どこへ向かっているかの両方を知ることはありません。マルチホップはデフォルトで無効になっており、アプリ内で正直に説明されています。

5. 支払い（プレミアムにアップグレードする場合のみ）

CumulusVPN には支払い不要、身元を特定する情報も不要な無料プランがあります。

プレミアム（より高速）にアップグレードすることを選んだ場合、Flux の公開ブロックチェーン上でFLUX 暗号資産 によって支払います。重要なポイントは以下のとおりです。

  • 支払いはウェブまたはデスクトップで行われ、モバイルアプリ内では行われません。モバイルアプリには 購入画面がなく、現在のプランを表示するだけです。
  • あなたは自分自身のウォレットから、私たちが公開している支払いアドレスへ FLUX を送金し、あなたの公開鍵から 導き出された短いメモを添えます。これによりネットワークは正しい鍵にクレジットを付与できます。私たちは カード番号、請求先住所、あなたの氏名を収集しません。
  • ブロックチェーン取引は本質的に公開されています。私たちを含む誰もが、公開 台帳上で、ある金額の支払いが特定のメモとともに私たちのアドレスに送られたことを確認できます。その台帳は Flux のブロックチェーンによって運営されており、私たちが運営しているわけではありません。私たちはその取引をあなたの実在の身元と結びつけることはなく、メモは あなたの公開鍵から一方向に導出されたものです——それはあなたの鍵やトラフィックを明かすものではありません。公開ブロックチェーンに 取引を公開することは本質的に公開行為であることをご理解のうえ、ご自身のウォレットのプライバシーにも十分ご注意ください。
  • 権利（ある鍵がプレミアムかどうか、いつまで有効か）は 公開ブロックチェーンから決定論的に算出され、公開鍵にのみ紐づけられます。私たちはあなたの氏名が記載された別個の顧客請求記録を保持していません。 なぜなら、私たちはあなたの氏名を持っていないからです。

もし将来、Apple の App Store や Google Play を通じて提供される法定通貨でのアプリ内支払いオプションをご利用いただく場合 （そのようなオプションを追加した場合）、その取引は Apple または Google が各社独自のプライバシーポリシーに基づいて 処理します。私たちはプレミアムを付与するために必要な確認情報のみを受け取り、あなたの支払いカードのデータは受け取りません。

6. サポートと任意の連絡

サポートのため、あるいは不正利用を報告するためにメールをいただいた場合、私たちはそのメールに書かれている内容——たとえば あなたのメールアドレスやメッセージの本文——を受け取ります。これはあなたへの返信のためだけに使用し、 問題解決に必要な期間を超えて保持することはありません。サービスの利用にあたって身元を明かす義務はなく、サポートへの連絡は任意です。

任意・オプトインのクラッシュレポート：もし——かつその場合に限り——あなたが明示的にクラッシュレポートを有効にした場合、 技術的な診断情報（アプリのバージョン、デバイスモデル、OS バージョン、スタックトレースなど）を含むクラッシュレポートが、 バグ修正のために送信されることがあります。これはデフォルトで無効であり、閲覧活動は一切含まれず、いつでもオフにできます。

7. Cookie とウェブサイト

CumulusVPN のウェブサイト（cumulusvpn.com）は、ダウンロード、ドキュメント、支払いページの提供、 および署名済みサーバーディレクトリのホスティングに使われる静的サイトです。広告用 Cookie や サードパーティの行動トラッカーは使用しません。使用される Cookie があるとすれば、それは純粋に機能的なもの（例：ライト/ダーク テーマの記憶）に限られます。

8. データの共有と開示

私たちは個人データを販売、貸与、あるいは取引することはありません——そもそも売るべきデータがほとんどありません。

私たちは活動ログもユーザーの身元情報も保持していないため、あるユーザーの閲覧行動に関するデータリクエストに対して 提供できるものが何もありません。法執行機関や裁判所から法的に有効な要請を受けた場合でも、 私たちが実際に保有している情報しか提供できず、そこにはユーザーの身元、閲覧履歴、 DNS クエリ、トラフィックの内容は含まれません。なぜなら、それらの記録がそもそも存在しないからです。適法かつ適切な場合には、受け取った 要請について記述した透明性レポートを公開する予定です。

私たちはユーザープロファイルを広告主、データブローカー、分析プロバイダーに提供することはありません。そもそも ユーザープロファイルを構築していないからです。

9. 国際的なデータ移転

CumulusVPN はグローバルなゲートウェイのネットワークです。あなたが接続すると、あなたのトラフィックは あなたが選んだサーバーの国から出ていきます。私たちは個人データを中央 データベースに集約していないため、保護すべき個人データストアの国境を越えた移転は発生しません。エグジット国の 選択はあなた自身が行うものであり、セッションごとに新たに選び直されます。

10. あなたの権利（GDPR / UK GDPR / CCPA など）

お住まいの地域によっては、あなたの個人データにアクセス、訂正、削除、 またはポータビリティを求める権利、処理に異議を唱えたり制限したりする権利、そして（カリフォルニア州では）何が収集されているかを知り、 これらの権利を行使したことで差別を受けない権利をお持ちの場合があります。サービスが匿名であるため：

  • アクセス／データポータビリティ：私たちは通常、あなたに紐づく個人データを保有していないため提供できません。 サポートにメールを送ったことがある場合は、そのやり取りのコピーを請求できます。
  • 削除：あなたの身元に紐づいてゲートウェイから削除すべきものは何もありません。ピア状態はもともと一時的な RAM のみであり、切断／再起動時に消去されます。あなたはいつでもデバイス上で鍵を削除でき、それにより仮名としてのつながりは完全に断ち切られます。
  • 販売しない／共有しない：私たちは個人データを販売も共有もしないため、オプトアウトすべき対象自体がありません。

権利を行使するには、info@cumulusvpn.com までメールをお送りください。私たちの（存在しない）記録の中であなたを 特定できない場合がありますが、それはまさにサービスが匿名として設計されているためです——これは設計であり、回避ではありません。あなたはまた、お住まいの地域のデータ保護当局に苦情を申し立てる権利も有しています。

11. お子様について

CumulusVPN は 13 歳未満のお子様（またはお住まいの国におけるデジタル同意の最低年齢）を対象としていません。 私たちはお子様から意図的に個人データを収集することはありません。誰からも識別情報を収集していないため、 利用者の年齢を判断することはできません。本サービスは一般の方を対象としており、お子様向けに販売促進を行うことはありません。

12. セキュリティ

あなたのデバイスとゲートウェイの間のトラフィックは、WireGuard® プロトコル （モダンで監査済みの暗号技術）を用いてエンドツーエンドで暗号化されます。あなたの秘密鍵はあなたのデバイスから決して外に出ません。サーバー側のピア状態 はメモリ上にのみ保持されます。アプリは各ゲートウェイの制御 API に直接接続してアクセスし、その応答は ed25519 で署名されアプリによって検証されるため、サーバーディレクトリとゲートウェイ応答の真正性は、トランスポート証明書ではなくデジタル署名によって保護されています。
