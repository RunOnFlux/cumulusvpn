1. สรุป (นโยบายทั้งหมดในหน้าจอเดียว)

  • ไม่มีบัญชี ไม่มีอีเมล ไม่มีรหัสผ่าน ไม่มีหมายเลขโทรศัพท์ คุณไม่เคยต้องมอบตัวตนของคุณให้เรา
  • ไม่มีล็อกกิจกรรม เราไม่บันทึกเว็บไซต์ที่คุณเข้าชม คำค้น DNS เนื้อหาทราฟฟิก ประวัติแบนด์วิดท์ หรือประทับเวลาการเชื่อมต่อของคุณลงในที่จัดเก็บถาวรใด ๆ
  • ตัวตนของคุณคือคีย์เข้ารหัส ที่สร้างขึ้นบนอุปกรณ์ของคุณเอง เราไม่มีทางเห็นคีย์ส่วนตัว ของคุณ เราเห็นเพียงคีย์สาธารณะ ซึ่งเป็นสตริงที่ดูสุ่มและไม่มีความเชื่อมโยงกับตัวคุณ
  • เกตเวย์เก็บสถานะ peer ไว้ใน RAM เท่านั้น เมื่อเซิร์ฟเวอร์รีสตาร์ท สถานะนั้นจะหายไป ไม่มีฐานข้อมูลทราฟฟิกให้ยึด เรียกหมายศาล หรือรั่วไหล
  • การชำระเงินเป็นทางเลือกและทำด้วยคริปโทเคอร์เรนซี FLUX บนบล็อกเชนสาธารณะ เรา ไม่ได้เป็นผู้ดำเนินการบล็อกเชน ไม่รับข้อมูลบัตร และไม่มีทางรู้ชื่อของคุณจากการชำระเงิน
  • ไม่มีตัวติดตาม ไม่มี SDK โฆษณา ไม่มีการวิเคราะห์ที่ระบุตัวตนคุณได้ แอปไม่มีโค้ด โฆษณาจากบุคคลที่สามหรือโค้ดติดตามพฤติกรรมใด ๆ
  • เราไม่ใช่ธุรกิจข้อมูล เราไม่มีข้อมูลผู้ใช้ให้ขาย และเราไม่ขายข้อมูล

2. เราคือใคร

CumulusVPN ("เรา") เป็นบริการ VPN ที่ดำเนินการโดยทีม CumulusVPN สร้างขึ้นบนเครือข่ายคลาวด์แบบกระจายศูนย์ของ Flux (runonflux.io) เซิร์ฟเวอร์ exit ของ VPN ("เกตเวย์") ทำงานเป็นแอปแบบ container บนโหนดศูนย์ข้อมูลของ Flux ที่ดำเนินการโดยผู้ให้บริการศูนย์ข้อมูลอิสระที่ผ่านการยืนยัน KYC เราเผยแพร่บริการนี้ ส่วน compute ที่อยู่เบื้องหลังนั้นกระจายศูนย์

สำหรับวัตถุประสงค์ของกฎหมายคุ้มครองข้อมูลส่วนบุคคลทั่วไปของสหภาพยุโรป/สหราชอาณาจักร (GDPR) ในขอบเขตที่จำกัดมากที่ข้อมูลด้านล่างนี้อาจถือเป็น "ข้อมูลส่วนบุคคล" นิติบุคคลผู้ดำเนินการ CumulusVPN คือผู้ควบคุมข้อมูล ฐานทางกฎหมายที่เราอ้างอิงคือผลประโยชน์อันชอบธรรมของเรา (มาตรา 6(1)(f)) ในการให้บริการ VPN ที่ใช้งานได้จริงและต้านทานการใช้งานในทางที่ผิด และ — สำหรับการชำระเงิน — การปฏิบัติตามบริการที่คุณร้องขอ (มาตรา 6(1)(b)) เนื่องจากบริการนี้ถูกออกแบบมาให้ไม่ระบุตัวตน ในกรณีส่วนใหญ่เราจึงไม่มีข้อมูล ที่สามารถเชื่อมโยงกับคุณได้เลย

3. สิ่งที่เราไม่เก็บ

เราต้องการพูดให้ชัดเจน เพราะ VPN ส่วนใหญ่มักฝังเรื่องนี้ไว้ CumulusVPN ไม่เก็บ บันทึก หรือจัดเก็บสิ่งต่อไปนี้เลย:

  • ชื่อ ที่อยู่อีเมล หมายเลขโทรศัพท์ หรือข้อมูลรับรองบัญชีใด ๆ ของคุณ — เพราะไม่มีบัญชี
  • IP ต้นทางของคุณในล็อกถาวรใด ๆ
  • เว็บไซต์ โดเมน หรือบริการที่คุณเชื่อมต่อไป
  • คำค้น DNS ของคุณ (DNS ถูก resolve ให้คุณผ่าน tunnel และไม่ถูกบันทึก)
  • เนื้อหา เวลา หรือขนาดของทราฟฟิกของคุณ ในฐานข้อมูลถาวรใด ๆ
  • ประวัติแบนด์วิดท์ ประวัติเซสชัน หรือประวัติประทับเวลาการเชื่อมต่อที่ผูกกับคุณ
  • ตัวระบุโฆษณาของอุปกรณ์ (IDFA / GAID) — เราไม่ร้องขอหรือใช้สิ่งเหล่านี้
  • ข้อมูลตำแหน่งจากอุปกรณ์ของคุณ (GPS แบบละเอียดหรือแบบคร่าว ๆ) ประเทศที่คุณเชื่อมต่อผ่าน คือตัวเลือกเซิร์ฟเวอร์ของคุณ ไม่ใช่การอ่านตำแหน่งอุปกรณ์ของคุณ
  • รายชื่อผู้ติดต่อ รูปภาพ ไฟล์ ไมโครโฟน กล้อง หรือเซนเซอร์ใด ๆ ของอุปกรณ์

เราไม่ใช้เครือข่ายโฆษณาจากบุคคลที่สาม และเราไม่ใช้ผลิตภัณฑ์วิเคราะห์ที่สร้างโปรไฟล์ของคุณ (เช่น SDK ด้าน ad-tech) แอปของเราไม่มีตัวติดตามพฤติกรรมติดตั้งมาด้วย

เว็บไซต์ของเรา (cumulusvpn.com และ vpn.cumulusvpn.com) ใช้ Cloudflare Web Analytics ซึ่งเป็นบริการวัดผลแบบไม่มีคุกกี้และเป็นข้อมูลรวม — ไม่ตั้งคุกกี้ ไม่จัดเก็บ สิ่งใดบนอุปกรณ์ของคุณ และไม่สามารถระบุตัวตนหรือสร้างโปรไฟล์ของผู้เข้าชมแต่ละคนได้ มันบอกเราแค่จำนวนหน้าที่ถูกดู ไม่ใช่ตัวบุคคล แอปของเราไม่มีระบบวิเคราะห์ใด ๆ ทั้งสิ้น

4. อะไรบ้างที่เกี่ยวข้องในทางเทคนิค (และเหตุใดจึงไม่ใช่โปรไฟล์ของคุณ)

VPN ไม่สามารถทำงานได้หากไม่มีข้อมูลบางอย่างไหลผ่านมันแบบเรียลไทม์ นี่คือทั้งหมดของมัน และเหตุผลว่าทำไม ไม่มีสิ่งใดกลายเป็นบันทึกเกี่ยวกับตัวคุณ

4.1 คีย์สาธารณะของคุณ (ตัวระบุแบบนามแฝง)

อุปกรณ์ของคุณสร้างคู่คีย์ WireGuard ในเครื่องตั้งแต่เปิดใช้งานครั้งแรก คีย์ส่วนตัวไม่เคย ออกจากอุปกรณ์ของคุณ — แม้แต่มาถึงเราก็ไม่ มีเพียงคีย์สาธารณะเท่านั้นที่ถูกส่งไปยัง เกตเวย์เพื่อให้มันกำหนดเส้นทาง tunnel ที่เข้ารหัสของคุณได้ คีย์สาธารณะคือค่าที่ดูสุ่มขนาด 32 ไบต์ มัน ไม่ผูกกับชื่อ อีเมล หรือฮาร์ดแวร์ของคุณ และคุณสามารถสร้างใหม่ได้ทุกเมื่อ มันคือนามแฝง ไม่ใช่ตัวตน

4.2 สถานะ peer ในหน่วยความจำบนเกตเวย์

ขณะที่คุณเชื่อมต่ออยู่ เกตเวย์จะเก็บ — ไว้ในหน่วยความจำชั่วคราว (RAM) เท่านั้น — สิ่งที่จำเป็น ขั้นต่ำในการกำหนดเส้นทางแพ็กเก็ต: คีย์สาธารณะของคุณ, IP ภายในชั่วคราวของ tunnel ที่ถูกกำหนดให้ (เช่น 10.8.x.y) และตัวนับไบต์ที่ใช้บังคับใช้ระดับความเร็วของคุณ สถานะนี้ไม่เคยถูกเขียนลง ดิสก์ ไม่ถูกส่งไปยังเซิร์ฟเวอร์กลางใด ๆ และจะถูกลบเมื่อโปรเซสของ เกตเวย์รีสตาร์ท หรือคุณตัดการเชื่อมต่อ ไม่มีบันทึกย้อนหลังที่สร้างจากสิ่งนี้

4.3 IP จริงของคุณ แบบชั่วคราว

ในการส่งทราฟฟิกของคุณ เกตเวย์จำเป็นต้องเห็น IP ที่การเชื่อมต่อของคุณมาจาก ณ ขณะนั้น ที่ระดับเครือข่าย — นี่คือวิธีที่เซิร์ฟเวอร์อินเทอร์เน็ตใด ๆ ทำงาน CumulusVPN ไม่บันทึกที่อยู่นี้ลง ที่จัดเก็บถาวรใด ๆ และไม่เชื่อมโยงมันกับการท่องเว็บของคุณ เมื่อเซสชันของคุณสิ้นสุด มันจะไม่ถูกเก็บไว้

4.4 DNS

การค้นหา DNS ถูก resolve แทนคุณผ่าน tunnel ที่เข้ารหัส ทำให้เครือข่ายท้องถิ่นและ ISP ของคุณ ไม่สามารถเห็นมันได้ เราไม่เก็บล็อกคำค้น DNS

4.5 แอปของคุณเข้าถึงเกตเวย์อย่างไร

แอปของคุณค้นหาเกตเวย์และอ่านสถานะของมันโดยเรียก control API ของแต่ละเกตเวย์โดยตรง คำตอบเหล่านั้นถูกเซ็นด้วยคีย์ ed25519 ของเกตเวย์และตรวจสอบโดยแอป ดังนั้นการปลอมแปลงจะถูกตรวจพบแม้ว่าช่องทางควบคุมเองจะเป็นการเชื่อมต่อธรรมดา — การรับประกัน ความสมบูรณ์มาจากลายเซ็น ไม่ใช่จากใบรับรอง ทราฟฟิกอินเทอร์เน็ตจริงของคุณแยกต่างหาก และเข้ารหัสแบบ end-to-end ด้วย WireGuard ระหว่างอุปกรณ์ของคุณกับเกตเวย์

4.6 ผู้ให้บริการเกตเวย์เห็นอะไรได้และไม่ได้

เกตเวย์ทำงานบนโหนดศูนย์ข้อมูลของ Flux เหมือนกับ VPN exit ทั่วไป ผู้ให้บริการที่สังเกตเซิร์ฟเวอร์ของตนเอง สามารถเห็น metadata การไหลที่เข้ารหัสแบบเรียลไทม์ (ว่า peer กำลังส่งทราฟฟิก) แต่ไม่ใช่บันทึกถาวร ที่เราสร้าง และไม่ใช่เนื้อหาของ tunnel สำหรับผู้ใช้ที่ไม่ต้องการให้เซิร์ฟเวอร์เดียวเห็นทั้ง IP และปลายทางของตน CumulusVPN มีโหมด multi-hop แบบเลือกใช้เอง: ทราฟฟิกของคุณจะ ถูกส่งผ่านสองเกตเวย์เพื่อไม่ให้เซิร์ฟเวอร์เดียวรู้ทั้งว่าคุณเป็นใครและคุณกำลัง ไปที่ไหน multi-hop ปิดใช้งานเป็นค่าเริ่มต้นและถูกอธิบายไว้อย่างตรงไปตรงมาในแอป

5. การชำระเงิน (เฉพาะเมื่อคุณอัปเกรดเป็นพรีเมียม)

CumulusVPN มีระดับฟรีที่ไม่ต้องชำระเงินและไม่ต้องมีข้อมูลระบุตัวตนใด ๆ

หากคุณเลือกอัปเกรดเป็นพรีเมียม (ความเร็วสูงขึ้น) คุณจะจ่ายด้วยคริปโทเคอร์เรนซี FLUX บนบล็อกเชนสาธารณะของ Flux ประเด็นสำคัญ:

  • การชำระเงินเกิดขึ้นบนเว็บหรือเดสก์ท็อป ไม่ใช่ในแอปมือถือ แอปมือถือ ไม่มีหน้าจอซื้อ มันแสดงเพียงระดับปัจจุบันของคุณเท่านั้น
  • คุณส่ง FLUX จากwallet ของคุณเองไปยังที่อยู่ชำระเงินที่เราเผยแพร่ไว้ พร้อมกับ memo สั้น ๆ ที่ได้มาจากคีย์สาธารณะของคุณ เพื่อให้เครือข่ายสามารถให้เครดิตกับคีย์ที่ถูกต้องได้ เรา ไม่เก็บหมายเลขบัตร ที่อยู่สำหรับเรียกเก็บเงิน หรือชื่อของคุณ
  • ธุรกรรมบนบล็อกเชนเป็นสาธารณะโดยธรรมชาติ ใครก็ตาม รวมถึงเราด้วย สามารถเห็นบน ledger สาธารณะว่ามีการชำระเงินจำนวนหนึ่งถูกส่งมายังที่อยู่ของเราพร้อม memo ที่กำหนด ledger นั้น ดำเนินการโดยบล็อกเชนของ Flux ไม่ใช่โดยเรา เราไม่เชื่อมโยงธุรกรรมนั้นกับตัวตนจริงของคุณ และ memo ถูกสร้างขึ้นแบบทางเดียวจากคีย์สาธารณะของคุณ — มันไม่เปิดเผยคีย์หรือทราฟฟิกของคุณ โปรดเข้าใจว่า การเผยแพร่ธุรกรรมไปยังบล็อกเชนสาธารณะนั้นเป็นสาธารณะโดยเนื้อแท้ กรุณาดูแลความเป็นส่วนตัวของ wallet คุณให้เหมาะสม
  • สิทธิ์การใช้งาน (ว่าคีย์หนึ่งเป็นพรีเมียมหรือไม่ และจนถึงเมื่อใด) ถูกคำนวณอย่างเป็นกำหนดแน่นอนจาก บล็อกเชนสาธารณะ และผูกกับคีย์สาธารณะเท่านั้น เราไม่จัดเก็บบันทึกการเรียกเก็บเงินลูกค้าแยกต่างหากที่มีชื่อของคุณอยู่ในนั้น เพราะเราไม่มีชื่อของคุณ

หากในภายหลังคุณใช้ตัวเลือกการชำระเงินแบบเงินจริงในแอปที่เปิดให้ผ่าน App Store ของ Apple หรือ Google Play (หากเราเพิ่มตัวเลือกนี้) ธุรกรรมนั้นจะถูกดำเนินการโดย Apple หรือ Google ภายใต้นโยบายความเป็นส่วนตัวของ พวกเขาเอง เราได้รับเพียงการยืนยันที่จำเป็นเพื่อมอบสิทธิ์พรีเมียม ไม่ใช่ข้อมูลบัตรชำระเงินของคุณ

6. การช่วยเหลือและการติดต่อโดยสมัครใจ

หากคุณส่งอีเมลหาเรา (เพื่อขอความช่วยเหลือหรือรายงานการใช้งานในทางที่ผิด) เราจะได้รับสิ่งที่คุณใส่ไว้ในอีเมลนั้น — ตัวอย่างเช่น ที่อยู่อีเมลของคุณและเนื้อหาข้อความ เราใช้มันเพียงเพื่อตอบคุณและเก็บไว้ไม่นานเกินกว่า ที่จำเป็นในการแก้ไขเรื่องนั้น คุณไม่จำเป็นต้องระบุตัวตนของคุณเพื่อใช้บริการ การติดต่อฝ่ายช่วยเหลือเป็นไปโดยสมัครใจ

รายงาน crash แบบเลือกเข้าร่วมโดยสมัครใจ: หาก — และเฉพาะเมื่อ — คุณเปิดใช้งานการรายงาน crash อย่างชัดแจ้ง รายงาน crash ที่มีข้อมูลวินิจฉัยทางเทคนิค (เช่น เวอร์ชันแอป รุ่นอุปกรณ์ เวอร์ชัน OS และ stack trace) อาจถูกส่ง เพื่อช่วยเราแก้บั๊ก ค่าเริ่มต้นคือปิดใช้งาน ไม่มีข้อมูลการท่องเว็บใด ๆ และสามารถปิดได้ทุกเมื่อ

8. การแชร์และเปิดเผยข้อมูล

เราไม่ขาย ให้เช่า หรือแลกเปลี่ยนข้อมูลส่วนบุคคล — เราแทบไม่มีข้อมูลใดให้ขายเลย

เนื่องจากเราไม่เก็บล็อกกิจกรรมหรือตัวตนผู้ใช้ใด ๆ เราจึงไม่มีอะไรให้ส่งมอบ เพื่อตอบสนองคำขอข้อมูลเกี่ยวกับการท่องเว็บของผู้ใช้ หากเราได้รับคำขอที่ถูกต้องตามกฎหมายจากหน่วยงานบังคับใช้กฎหมาย หรือคำสั่งศาล เราให้ได้เพียงข้อมูลที่เรามีอยู่จริงเท่านั้น ซึ่งไม่รวมตัวตนผู้ใช้ ประวัติการท่องเว็บ คำค้น DNS หรือเนื้อหาทราฟฟิก เพราะบันทึกเหล่านั้นไม่มีอยู่จริง ในกรณีที่ถูกกฎหมายและเหมาะสม เราตั้งใจ จะเผยแพร่รายงานความโปร่งใสที่อธิบายคำขอที่ได้รับ

เราไม่โอนโปรไฟล์ผู้ใช้ให้กับผู้ลงโฆษณา data broker หรือผู้ให้บริการวิเคราะห์ เพราะเราไม่ สร้างโปรไฟล์ผู้ใช้ตั้งแต่แรก

10. สิทธิของคุณ (GDPR / UK GDPR / CCPA และที่คล้ายกัน)

ขึ้นอยู่กับว่าคุณอาศัยอยู่ที่ไหน คุณอาจมีสิทธิเข้าถึง แก้ไข ลบ หรือโอนย้ายข้อมูลส่วนบุคคลของคุณ คัดค้านหรือจำกัดการประมวลผล และ (ในแคลิฟอร์เนีย) รับทราบว่ามีการเก็บอะไรบ้างและไม่ถูกเลือกปฏิบัติจากการใช้สิทธิเหล่านี้ เนื่องจากบริการนี้ไม่ระบุตัวตน:

  • การเข้าถึง / การโอนย้ายข้อมูล: โดยทั่วไปเราไม่มีข้อมูลส่วนบุคคลที่เชื่อมโยงกับคุณให้มอบให้ได้ หากคุณเคยส่งอีเมลหาฝ่ายช่วยเหลือ คุณสามารถขอสำเนาของการติดต่อสื่อสารนั้นได้
  • การลบข้อมูล: ไม่มีอะไรผูกกับตัวตนของคุณให้เราลบออกจากเกตเวย์ สถานะ peer เป็นแบบ RAM ชั่วคราวอยู่แล้ว และจะถูกล้างเมื่อตัดการเชื่อมต่อ/รีสตาร์ท คุณสามารถลบคีย์ของคุณบนอุปกรณ์ได้ทุกเมื่อ ซึ่งจะตัดการเชื่อมโยงแบบนามแฝงอย่างสมบูรณ์
  • ไม่ขาย / ไม่แชร์ข้อมูล: เราไม่ขายหรือแชร์ข้อมูลส่วนบุคคล จึงไม่มีอะไรให้เลือกไม่รับ

หากต้องการใช้สิทธิใด ๆ ให้ส่งอีเมลไปที่ info@cumulusvpn.com เราอาจไม่สามารถระบุตัวตนของคุณได้ใน บันทึก (ที่ไม่มีอยู่) ของเรา ก็เพราะบริการนี้ถูกออกแบบให้ไม่ระบุตัวตน — นั่นคือการออกแบบ ไม่ใช่การหลบเลี่ยง คุณยังมีสิทธิยื่นเรื่องร้องเรียนต่อหน่วยงานคุ้มครองข้อมูลในพื้นที่ของคุณได้ด้วย
